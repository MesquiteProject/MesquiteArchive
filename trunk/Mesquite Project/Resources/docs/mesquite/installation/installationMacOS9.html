Mesquite installation for MacOS 9

(Please email us () with questions or comments about downloading Mesquite).

Installation:

  1. Download this file: mesquite.sit (Stuffit)
  2. Double click mesquite.sit; this should start StuffitExpander and unstuff the file. If you don't have StuffitExpander you can download a free version at www.stuffit.com. This will create a folder approximately as follows:

  3. Double click the Mesquite icon to start Mesquite.

Requirements: MRJ 2.2 or better (www.apple.com/java)

NOTE: If you have a previous version of Mesquite installed in your system, we advise that you delete it before running the new version.

NOTE: Mac OS 9 does not support modern versions of Java. Therefore some Mesquite features will be missing under Mac OS 9.
