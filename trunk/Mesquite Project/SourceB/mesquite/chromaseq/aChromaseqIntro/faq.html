Frequently Asked Questions

 

Q: I have used Chromaseq and Phred and Phrap to produce a consensus sequence. I've also made more base calls using Chromaseq's chromatogram viewer. However, one part of that sequence isn't the best, and I have resequenced part of the gene. Can I add these new chromatogram to the existing contig?

A: No. You will need to reprocess the original chromatograms again for that sequence, and remake any base calls. Sorry. If you want these sorts of tools, you will need to use dedicated chromatogram manipulation packages (see Sequencher & Others).

 

Q: Can I insert bases into a sequence?

A: Not yet. We hope to add this soon.

 

Q: I would like to use Chromaseq to make my base calls, then export the matrix as an NBRF file, read it into an alignment program, reimport the aligned matrix back into Mesquite, but still be able to look at the chromatograms for the sequences in this aligned matrix. Can I do this?

A: Not yet.
