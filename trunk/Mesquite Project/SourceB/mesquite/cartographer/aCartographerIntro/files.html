Saving Graphics Files

 

The best way to save your map as a graphics file is by saving it as a PDF or PostScript file, and then edit it in Adobe Illustrator or a similar program that deals with full vector graphics. If you do this, the symbols will appear in high resolution, much higher than they appear on screen. Saving files as PDF or PostScript can be done as follows:

  • MacOS: MacOS X and MacOS 9 have built into them the ability to save images as PDF files. In MacOS X, ask to print the taxa or tree window containing the map using File>Print to Fit Page. In the Print dialog box that appears, choose the Save As PDF button.
  • Windows: One solution is to download and install an Adobe Postscript Printer Driver, which should allow you set your printer so that when you ask to ask to print the taxa or tree window containing the map using File>Print to Fit Page, the image will be printed to a PostScript file rather than a physical printer.

Make sure that your base map is of high enough revolution for your publication needs.
