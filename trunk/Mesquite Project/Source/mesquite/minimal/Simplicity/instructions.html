Simple vs. Full Interface

To simplify Mesquite's interface by hiding many menu items and buttons, choose Simple Interface in the panel above or in the info bars at the top of most windows.

To use Mesquite's full interface with all menu items and buttons, choose Full Interface in the panel above or in the info bars at the top of most windows.

To configure what is hidden when you choose the simple mode, hit the button Turn ON Editing Mode in the panel above. For instructions of how to use the Editing Mode, see below.

What is hidden in Simple mode?

A menu item or tool is hidden if it has been specifically chosen to be hidden, or if the Mesquite package that controls it has been hidden.

To hide all the menu items and tools belonging to a Mesquite package or module, uncheck its box in the Simplify Control Panel here. The list of packages and modules to be hidden can be accessed only when in Editing Mode. To edit this list, Turn ON Editing mode by hitting the button above, but then you'll have to turn it off again to continue reading here.

To hide a single menu item or tool, select it when Editing Mode is on. In Editing Mode, when you select menu items or tools, they will be highlighted to show they will be hidden. Hidden menu items will be marked with "(OFF)" if you have chosen that particular item to hide, "(PACKAGE OFF)" if the item will be hidden because its package has been hidden. Hidden tools will be ringed in red if you have chosen that particular item to hide, in blue if hidden because its package has been hidden.

After you have edited what you want to hide, hit the button above "Turn OFF Editing Mode". Then, choose "Simple Interface" to show the simplified interface.

Saving and loading simplifications

You can save your current set of choices for hiding packages, menu items and tools by selecting Save Current Simplification in the drop down menu labeled Save/Load in the Simplify Control Panel above.

You can load pre-existing settings for simplifications by selecting items from the Load Simplification submenu of the same drop down menu.

The pre-existing simplifications are stored in Mesquite_Folder/settings/mesquite/minimal/Simplify. If you want to customize and store your own simplification among the pre-existing ones, copy Simplification.xml from Mesquite_Support_Files/Mesquite_Prefs into Mesquite_Folder/settings/mesquite/minimal/Simplify. You can rename this file, and you can also go into the file with a text editor and change "<name>Custom</name>" to a name your prefer, for example "<name>For Students</name>".
