Overview

This package calculates consensus trees similarity measures between trees. IT HAS FEW MODULES, WHICH HAVE NOT BEEN WELL TESTED. It is hoped that other programmers will decide to build packages for these purposes.

Modules

The modules currently included are:

  • ConsensusTree -- Supervises the calculating of consensus trees; serves as a source of trees
  • ConsensusTreesFrom -- Supervises the calculating of consensus trees; serves as a tree block filler
  • SemistrictConsensus -- calculates something approximating a semistrict consensus of a set of trees
  • SharedClades -- counts the number of shared clades between two rooted trees
  • SharedPartitions -- counts the number of shared partitions between two trees (an unrooted measure)

Installation

To install the Tree comparison package, the "treecomp" directory must be installed in the "mesquite" directory within "Mesquite Folder".
