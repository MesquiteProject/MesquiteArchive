Sources of Base Maps

There are a number of sources of base maps or images of the earth that are available for general use. These include:

  • NASA's Planetary Photojournal
    • World
    • Africa
    • Spain
    • South America
    • North America Natural Color Mosaic
    • North America Shaded Relief
    • Central America
    • New Zealand
    • Australia
  • Perry-Castañeda Library Map Collection

Some base maps are distributed with Cartographer in the "maps" directory. These maps contain, in their directories, a readMe file with details about the map's projection and copyright information, and an XML file containing parameter and calibration information which can be loaded into Mesquite once the projection has been set.
