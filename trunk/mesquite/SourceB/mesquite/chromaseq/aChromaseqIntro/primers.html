Primer File

In order for Chromaseq to be able to process the chromatogram files so that Phred and Phrap can analyze them, Chromaseq needs to know the names of the primers associated with the various gene fragments or genes, and the relative direction of the primers.

This information is provided to Chromaseq in the form of a file listing the fragments and the primers for each fragment. The format of this text file is as follows:

<fragment name> Forward <forward primer name> <forward primer name> Reverse <reverse primer name> <reverse primer name>;
<fragment name> Forward <forward primer name> <forward primer name> Reverse <reverse primer name> <reverse primer name>;

Here is an example file:

28S Forward D1 D2F 28s1a D62A '28S4.8a' '28S4-8a' Reverse D3 D2R 28SB 28S7b1 D61b D62B;
COI Forward Jer Jerry Dick B1490 B1499 B1513 B2679 Reverse Pat PatB B2232 Bcoi2R;

Note that "Forward" and "Reverse" are not allowed as primer names.
