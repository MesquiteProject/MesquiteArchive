Chromaseq and other programs

Sequencher

Instead of using Mesquite's built-in chromatogram viewer, you may wish to use a program such as GeneCode's Sequencher™. This program can read the .ace files produced by Phred/Phrap/Mesquite and display the traces, consensus sequences, quality scores, etc.

Advantages of using the Phred/Phrap/Mesquite system for producing your initial contigs for Sequencher are:

  • entire directories of chromatograms will be reorganized into gene fragments and samples
  • contigs will be renamed by Mesquite
  • base quality scores will be visible within Sequencher

If you have particularly difficult reads, that are not easily or accurately assembled into contigs by Phrap, you will either need to adjust Phred and Phrap's action by learning their command structure and modifying their default behavior, or you will need to use another program like Sequencher to process those sequences. Advantages of using Sequencher rather than Chromseq's chromatogram viewer for this purpose are:

  • interactive assembly of contigs is available
  • you can incorporate additional reads later without redoing the Phred/Phrap/Mesquite analysis

Aligner

CodonCode's Aligner™ program might also be used in a similar fashion to Sequencher. Aligner uses Phred and Phrap directly. We have no experience using it.

Consed

Consed also uses Phred and Phrap directly. We have limited experience using it.
