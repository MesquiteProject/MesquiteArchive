Overview

This page represents the JAMA package, as installed for use in Mesquite. When installed, various modules in Mesquite have the option of using JAMA methods for calculations. To find out what modules might be using JAMA, look through their documentation.

For more information about JAMA see http://math.nist.gov/javanumerics/jama/

Installation

To install Jama, the "Jama" directory must be installed within "Mesquite_Folder" directory.
